# Запуск и тесты
Чтобы собрать, запустите в корне
```bash
cmake -B build && cd build && make
```
Вы окажетесь в директории build, в которой будет находится бинарный файл `main`. Для запуска тестов необходимо запустить его с флагом `--test`
